Redefine Champion Select

Do you hate the music that plays during champion select and would rather listen to your own tunes? With this small application you can listen to your own music and enjoy selecting a champion!

Download .zip Download .tar.gz View on GitHub

Written completely in Ruby!

Cross-Platform! -- Only have to update the program with the path to your Game Folder!

Features to come:

  • Specify a Specific Music Folder with which the program will work.
  • Define which queue modes you'd like to have the background selection music change for!
  • Specify a specific song, or just have the program randomly select from the array of tracks!

Features to come:

  • Update for new champion select, currently only works for Random Game Mode + ARAM
  • Enable Looping for songs that don't fit the required length
  • Create a pop-up displaying the current track being played
  • Enable real-time music selection (option to move to next track if current one is not favorable)
